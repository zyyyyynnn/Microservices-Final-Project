## 指标口径

- `Total P95` 和 `Total TPS` 来自 JMeter 全部样本聚合，包含登录、秒杀受理、结果轮询和最终校验。
- 本摘要未生成独立 Transaction Controller 父样本，因此不记录“秒杀完整链路 P95”或“最终订单 TPS”。
- 原始 JTL 与 HTML Dashboard 是本地产物，不提交仓库；脱敏可复核数字见同目录 `seckill-ladder-20260610-234029.csv`。

## 停止边界

本轮未触发以下停止条件：

- HTTP 5xx；
- 非预期失败样本；
- 最终业务失败；
- 结果轮询超时；
- 重复 `orderNo`；
- 成功订单数大于 100；
- `order_info` 与 `order_item` 数量不一致；
- `locked` 与成功订单数不一致；
- Redis 库存小于 0；
- `Duplicate entry`、`Global lock wait timeout`、`RECONSUME`、死信、秒杀订单创建失败或结果回写失败日志命中。
